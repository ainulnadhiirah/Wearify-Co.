# Wearify-Co.
1. Project Title
Wearify Co. – One-Stop Fashion Haven Web System

2. Project Description
Wearify Co. is a modern online fashion store designed to deliver a smooth, stylish, and user-friendly shopping experience. The website showcases multiple fashion categories—Men, Women, Kids, and Accessories—through a clean layout, responsive design, and visually appealing slideshow banners. Created as a front-end fashion shopping platform, Wearify Co. aims to provide customers with convenience, clarity, and comfort while browsing products.

3. Features

a. User Interface Features

-Modern and responsive website layout
-Clean navigation bar with active page highlighting
-High-quality hero image slider (with arrows)
-Organized product categories (Men, Women, Kids, Accessories)
-Fashion-themed color palette for professional branding

b. Functional Features

-Homepage slideshow with promotional banners
-About Us page describing brand identity & vision
-Dashboard section
-Products listing page
-Categories filtering
-Logout button
-Login system (redirects if not logged in)

c. User Experience Enhancements

-Smooth image transitions
-Easy navigation between sections
-Consistent typography (Google Fonts)
-Mobile-responsive design
